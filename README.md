# The Making of a Scavenger Landing Page

Static cinematic landing page for Victor Syrota's prequel novella.

## Deploy

Upload the contents of this folder to any static host. The entry point is `index.html`.

## Storefront Links

The retailer buttons in `index.html` currently use placeholder `#` links. Replace them with the final Amazon, Apple Books, and Kobo URLs before launch.

The page uses `assets/cover.png` as the hero book visual and `assets/panorama.jpg` as the atmospheric world artwork.
